# Kit Documentation

Documentation for the Claude Code Workflow Kit itself — how to
install it, how the workflow is shaped, and how each skill works.

These docs live in the **kit repository only**. They are not copied
into target projects. A target project gets its own,
project-specific docs generated by the kit's skills (see
[`repo-structure.md`](repo-structure.md) for what goes where).

## Where to start

| File | Purpose |
|---|---|
| [`install.md`](install.md) | How to install the kit into a new target project |
| [`workflow-guide.md`](workflow-guide.md) | End-to-end walkthrough from idea or PRD to tagged release, plus the steady-state iteration loop |
| [`workflow-control.md`](workflow-control.md) | The control model: operating modes, the canonical approval gate, the human-facing verb layer, and how `/start` finds the next step |
| [`skills.md`](skills.md) | Functional reference for every skill in the kit, grouped by what they do in the workflow |

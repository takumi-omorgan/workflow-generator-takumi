## How to use this index

- Click an ADR to read its decision and consequences.
- Status reflects the current state: `proposed`, `accepted`,
  `rejected`, or `superseded by ADR-NNN`.
- Phase reflects which build-out-plan phase the ADR belongs to (or
  `—` for cross-cutting ADRs in this example: none).
